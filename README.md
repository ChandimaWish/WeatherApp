# Weather Application with Dynamic Wallpaper Changer

This is a Python application that utilizes the OpenWeather API to retrieve weather data for a specific location and changes the wallpaper accordingly on Windows. The program also uses the Unsplash API to download wallpapers. This project serves as an example of integrating APIs and using them in a practical application.

Configuration
In the config.py file, you will find the following variables that you can configure:

OPENWEATHER_API_KEY: Your OpenWeather API key.
UNSPLASH_API_KEY: Your Unsplash API key.
UNSPLASH_WALLPAPER_DIR: The directory to save the downloaded wallpapers.
